Pour aller plus loin#

Autre tutoriel#

Au délà de ce premier tutoriel, vous pouvez également celui de de Software Carpentry : Version Control with Git.

Ressouces compémentaires#

  1. Une introduction très pédagogique et détaillée de Sébastien Saunier : Débuter avec Git et Github (1h27)

  2. Des articles scientifiques sur l’utilisation de git et GitHub en recherche :

    • A Quick Introduction to Version Control with Git and GitHub, Blischak, Davenport & Wilson, PLOS Comput Biol, 2016.

    • Ten Simple Rules for Taking Advantage of Git and GitHub, Perez-Riverol et al., PLOS Comput Biol, 2016

    • Democratic databases: science on GitHub, Perkel, Nature, 2016.

  3. Les articles du blog de Christophe Ducamp :

    • GitHub Pour les Nuls : Pas de Panique, Lancez-Vous ! (Première Partie)

    • GitHub pour les Débutants : Consignez, Poussez et Foncez !

  4. La page Reproducible workflow and version control with Git and Github de Julie Lowndes, qui détaille comment configurer RStudio pour qu’il utilise Git et GitHub. Le site Happy Git and GitHub for the useR est également un site de référence pour combiner git, GitHub et RStudio.

  5. Enfin, la page dédiée à Git et GitHub de votre humble serviteur.
